# Platform Services

Landing page for the Platform Team's AWS service catalogue.

**Live site:** https://bv90dsit.github.io/platform-services/

## What this is

A GitHub Pages site (Jekyll) that helps tenants:

- Discover what AWS services the platform team offers
- Understand guardrails and constraints
- Onboard to the platform
- Raise questions or request new features via GitHub Issues

## Contributing

To run locally:

```bash
bundle install
bundle exec jekyll serve
```

Then visit http://localhost:4000/platform-services/
